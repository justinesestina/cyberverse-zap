# CyberVerse Explorer

An educational, responsive cybersecurity platform for beginners to explore hacker types, professional roles, and career paths.
Built with **React**, **Tailwind CSS**, and **Framer Motion**.

## Features

* Interactive Hacker Hat cards (White, Black, Grey, Blue, Red, Green)
* Detailed career modals with safe code examples
* Interactive career roadmap
* Multiple-choice quiz with personalized recommendations
* Dark cyber theme with neon gradients and smooth animations
